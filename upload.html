Accepted File Formats:

BioGeneXplorer supports FASTA and TXT file formats for uploads. Users should name files in a way that reflects their analysis purpose to ease reference in output files.

Upload Buttons:

Whole Genome Sequences (WGS): Upload WGS files under this button for analysis against sample genes.
Sample Genes: Upload target gene sequences under this button to initiate analysis against WGS data.

Setting Analysis Parameters:

Two adjustable parameters, Coverage and Identity, are available on the upload page for users to define analysis cutoffs. These are initially set to 90% but can be adjusted as desired.
